Trail Mix Lab — W05 Site Plan

Site plan outlining subject, purpose, audience scenarios, color schema, typography, and wireframes for the Week 05 milestone.

1) Site Name

Trail Mix Lab — a simple, memorable name that matches the core action: experimenting with ingredients to build a custom trail mix.

2) Site Purpose

  • Let users browse a catalog of trail mix ingredients with price, nutrition, allergens, and origin.
  • Provide filters and sorting to quickly find items that fit dietary needs and budgets.
  • Allow users to assemble and save a personal mix with live weight and cost totals, then send a message via a contact form.

3) Scenarios

  • Allergy‑aware hiker: “I’m allergic to peanuts — can I filter the list to exclude peanut and tree‑nut items?”
  • Budget planner: “I need about 500g of mix for a weekend trip and want to stay under $8 — which ingredients fit?”
  • Nutrition seeker: “Which options are vegan and higher in protein with moderate sugar?”

4) Color Schema

The palette balances warmth (trail foods) with clarity and accessible contrast. The same variables will be used in the site.

Cocoa — #6d4c41
Brand color: headers, primary buttons, active states.
Honey — #ffca28
Accent: badges, highlights, figure captions.
Link Blue — #0b57d0
Links: inline links, interactive text elements.
Ink — #1b1b1f
Text: body copy and icons on light backgrounds.

Contrast: body text uses Ink on white; accent backgrounds use dark text to meet WCAG AA.

5) Color Schema Use in This Document

Headings use Cocoa (brand), links use Link Blue, and figure captions use the Honey accent. The sections themselves show a Cocoa left border.

6) Typography: Named and Used

Primary font: system-ui stack (system-ui, -apple-system, Segoe UI, Roboto, Ubuntu, Cantarell, Helvetica Neue, Arial, sans-serif).

  • Body/UI: system-ui, regular weight.
  • Headings: system-ui, bold (600–700).
  • Code samples: ui-monospace, SFMono-Regular, Consolas, monospace (as fallback where relevant).
